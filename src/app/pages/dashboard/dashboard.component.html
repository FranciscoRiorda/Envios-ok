- Otras operaciones.
Asistente virtual
- Podrás realizar la carga de un nuevo Envío de paquetería siguiendo la ruta: Gestión de Pedidos > Envíos > 'Cargar nuevo Envío'.
Asistente virtual
- Podrás realizar la carga de un nuevo Gasto siguiendo la ruta: Gestión de Pedidos > Gastos > 'Cargar Gasto'.
Asistente virtual
- Podrás Actualizar o Modificar datos de un Envío cargado, siguiendo la ruta: Gestión de Pedidos > Envíos > . También podrás cambiar el estado de 'Retiro' y 'Entrega' de cada paquete desde la lista de pedidos.
Asistente virtual
- Podrás Actualizar o Eliminar un gasto, siguiendo la ruta: Gestión de Pedidos > Gastos. Podrás modificar el detalle del gasto desde su propio campo y luego deberás presionar el botón de Guardar que se encuentra a su derecha. Desde la podrás eliminar el gasto. Ten en cuenta que al eliminar el gasto, no podrá recuperarse.
Asistente virtual
- Mantenimientos de Clientes y Usuarios: Crear nuevo Cliente ó Crear nuevo Usuario.
- También podrás: Ver datos del Cliente ó Eliminar un cliente - Ver datos del Usuario , Cambiar contraseña del usuario ó Eliminar un usuario
Asistente virtual
- Para crear un nuevo Cliente, deberás seguir la ruta Mantenimientos > Clientes > 'Crear nuevo cliente'. Allí se te solicitarán los datos necesarios para dar de alta un nuevo cliente en el sistema.
Asistente virtual
- Para crear un nuevo Usuario, deberás seguir la ruta Mantenimientos > Usuarios > 'Crear nuevo usuario'. Allí se te solicitarán los datos necesarios para dar de alta un nuevo usuario en el sistema. Esta opción sólo estará disponible para usuarios Administradores.
Asistente virtual
- Podrás ver los datos de un cliente ingresando en Mantenimientos > Clientes > Datos del Cliente . Allí encontrarás los datos del cliente seleccionado. También podrás modificarlos en caso de ser necesario.
Asistente virtual
- Podrás eliminar un cliente ingresando en Mantenimientos > Clientes > Borrar . Ten en cuenta que al borrar el cliente, no podrá recuperarse.
Asistente virtual
- Podrás ver los datos de un usuario ingresando en Mantenimientos > Usuarios > Datos del Usuario . Desde el ícono de la derecha. Allí encontrarás los datos del Usuario seleccionado. También podrás modificarlos en caso de ser necesario. Esta opción sólo estará disponible para usuarios Administradores.
Asistente virtual
- Podrás modificar la contraseña de acceso del usuario ingresando en Mantenimientos > Usuarios > Cambiar Contraseña . Esta opción sólo estará disponible para usuarios Administradores.
Asistente virtual
- Podrás eliminar un usuario ingresando en Mantenimientos > Usuarios > Borrar . Ten en cuenta que al borrar el usuario, no podrá recuperarse. Esta opción sólo estará disponible para usuarios Administradores.
